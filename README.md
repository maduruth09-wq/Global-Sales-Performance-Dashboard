# Global Sales Performance Dashboard

## Project Overview
This project is an interactive Power BI dashboard designed to analyze sales performance across different countries, products, and customer segments. It provides a clear view of key business metrics such as total profit, sales, discounts, and product performance.

The dashboard helps users identify top-performing products, compare sales across countries, monitor discount trends, and track overall business performance through interactive charts and filters.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization

## Key Insights
- Paseo was the top-selling product.
- The USA generated the highest sales.
- Higher discount bands led to larger discount amounts.
- Government and Enterprise segments contributed significantly to sales.
